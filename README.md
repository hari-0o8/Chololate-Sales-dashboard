# Chololate-Sales-dashboard
**🍫 Chocolate Sales Analytics Dashboard
> 1. Project Title
Chocolate Sales Analytics Dashboard – Power BI
An interactive Power BI dashboard designed to analyze chocolate sales performance, profitability, shipments, product trends, and salesperson performance across different regions.
---
> 2. Short Description
This dashboard provides a comprehensive overview of chocolate sales data through interactive visualizations and KPIs. It helps business users monitor revenue, profit, shipment distribution, top-performing products, and salesperson contributions to support data-driven decision making.
---
> 3. Tech Stack
The dashboard was built using:
- 📊 Power BI Desktop
- 🔄 Power Query
- 📈 DAX (Data Analysis Expressions)
- 🗂 Data Modeling
- 📑 Excel Dataset
- 🖼 PNG Icons and Custom Formatting
---
> 4. Data Source
The dataset contains information related to:
- Sales Amount
- Profit
- Shipment Count
- Total Boxes
- Products
- Salespersons
- Locations/Regions
- Dates
Data was imported from Excel and transformed using Power Query.
---
> 5. Features / Highlights
>>> Business Problem
Sales managers need a quick way to monitor overall business performance, identify profitable products, track shipments, and evaluate salesperson contributions.
>>> Goal of Dashboard
- Track revenue and profitability
- Monitor shipment activity
- Analyze product performance
- Evaluate salesperson performance
- Compare regional sales distribution
>>> Key Visuals
=> KPI Cards
Displays:
- Total Amount
- Total Profit
- Profit Percentage
- Shipment Count
- Total Boxes
=> Amount CY vs PY
Compares Current Year sales against Previous Year sales.
=> Boxes CY vs PY
Compares box sales trends between years.
=> Shipment Distribution
Shows shipment volume distribution over time.
=> Top 6 Products
Highlights best-performing products based on sales.
=> Top 6 Salespersons
Ranks salespersons by sales amount and profit contribution.
=> Product Types Analysis
Displays amount and profit percentage for each product.
=> Amount by Geo
Regional sales distribution using a donut chart.
---
> 6. Business Insights
- Identifies top-performing products.
- Tracks profit margins across product categories.
- Helps recognize high-performing salespersons.
- Supports inventory and shipment planning.
- Provides geographical sales insights.
